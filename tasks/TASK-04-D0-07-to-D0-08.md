## Kết quả triển khai

### D0-07 Domain + SSL
- DNS kiểm tra từ VPS:
  - `sspaitools.com` -> `180.93.136.56`
  - `www.sspaitools.com` -> `180.93.136.56`
- Tạo Nginx vhost:
  - file `/etc/nginx/conf.d/sspaitools.conf`
  - `server_name sspaitools.com www.sspaitools.com`
- Cài certbot bằng Python pip (do AlmaLinux 10 repo không có package certbot trực tiếp)
- Cấp SSL thành công:
  - cert path: `/etc/letsencrypt/live/sspaitools.com/fullchain.pem`
  - key path: `/etc/letsencrypt/live/sspaitools.com/privkey.pem`
  - expiry: `2026-07-16`
- Verify:
  - `https://sspaitools.com` trả `HTTP/1.1 200 OK`
  - `https://www.sspaitools.com` trả `HTTP/1.1 200 OK`
  - `certbot renew --dry-run` thành công
